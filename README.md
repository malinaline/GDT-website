## Run Locally

Clone the project

```bash
  git clone https://github.com/malinaline/GDT-website.git
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start on Localhost

```bash
  npm start
```


## Tech Stack

**Client:** React, React-Bootstrap

**Server:** Node, Express, MongoDB
